수학 및 통계 함수

SUM
=SUM(숫자1, [숫자2], ...)
선택한 셀 범위의 합계를 계산합니다.
예: =SUM(A1:A10) - A1부터 A10까지의 셀 값을 모두 더합니다.
AVERAGE
=AVERAGE(숫자1, [숫자2], ...)
선택한 셀 범위의 평균값을 계산합니다.
예: =AVERAGE(B1:B20) - B1부터 B20까지의 평균값을 계산합니다.
COUNT
=COUNT(값1, [값2], ...)
선택한 범위에서 숫자가 포함된 셀 개수를 계산합니다.
예: =COUNT(C1:C30) - C1부터 C30까지 숫자가 있는 셀의 개수를 세어줍니다.
MAX
=MAX(숫자1, [숫자2], ...)
선택한 범위에서 최대값을 찾습니다.
예: =MAX(D5:D15) - D5부터 D15까지의 셀 중 가장 큰 값을 반환합니다.
MIN
=MIN(숫자1, [숫자2], ...)
선택한 범위에서 최소값을 찾습니다.
예: =MIN(E1:E100) - E1부터 E100까지의 셀 중 가장 작은 값을 반환합니다.
ROUND
=ROUND(숫자, 자릿수)
지정된 자릿수로 숫자를 반올림합니다.
예: =ROUND(A1, 2) - A1의 값을 소수점 2자리까지 반올림합니다.
SUMIF
=SUMIF(범위, 조건, [합계_범위])
조건을 만족하는 셀들의 합계를 계산합니다.
예: =SUMIF(A1:A10, ">100", B1:B10) - A1:A10에서 100보다 큰 값을 가진 셀에 대응하는 B1:B10 범위의 값들을 합산합니다.
COUNTIF
=COUNTIF(범위, 조건)
조건을 만족하는 셀의 개수를 계산합니다.
예: =COUNTIF(C1:C20, "완료") - C1:C20 범위에서 "완료"라는 텍스트가 있는 셀의 개수를 세어줍니다.
STDEV
=STDEV(숫자1, [숫자2], ...)
표본을 기준으로 하는 표준 편차를 계산합니다.
예: =STDEV(F1:F50) - F1:F50 범위의 숫자들의 표준 편차를 계산합니다.
POWER
=POWER(숫자, 지수)
숫자의 거듭제곱값을 반환합니다.
예: =POWER(2, 3) - 2의 3제곱인 8을 반환합니다.

텍스트 함수

CONCATENATE
=CONCATENATE(텍스트1, [텍스트2], ...)
여러 텍스트 문자열을 하나로 결합합니다.
예: =CONCATENATE("안녕하세요, ", A1, "님!") - "안녕하세요, "와 A1 셀의 값과 "님!"을 결합합니다.
LEN
=LEN(텍스트)
텍스트 문자열의 길이(문자 수)를 반환합니다.
예: =LEN(A1) - A1 셀에 있는 텍스트의 길이를 반환합니다.
LEFT
=LEFT(텍스트, [문자_수])
텍스트의 왼쪽에서부터 지정된 문자 수만큼 추출합니다.
예: =LEFT(A2, 3) - A2 셀의 텍스트 중 왼쪽에서 3자를 반환합니다.
RIGHT
=RIGHT(텍스트, [문자_수])
텍스트의 오른쪽에서부터 지정된 문자 수만큼 추출합니다.
예: =RIGHT(B1, 4) - B1 셀의 텍스트 중 오른쪽에서 4자를 반환합니다.
MID
=MID(텍스트, 시작_위치, 문자_수)
텍스트에서 지정된 위치에서 시작하여 지정된 문자 수만큼 추출합니다.
예: =MID(C1, 3, 5) - C1 셀의 텍스트 중 3번째 위치에서 시작하여 5자를 반환합니다.
UPPER
=UPPER(텍스트)
모든 문자를 대문자로 변환합니다.
예: =UPPER(D1) - D1 셀의 텍스트를 모두 대문자로 변환합니다.
LOWER
=LOWER(텍스트)
모든 문자를 소문자로 변환합니다.
예: =LOWER(E1) - E1 셀의 텍스트를 모두 소문자로 변환합니다.
TRIM
=TRIM(텍스트)
텍스트의 양쪽 끝 공백을 제거합니다.
예: =TRIM(F1) - F1 셀의 텍스트에서 불필요한 공백을 제거합니다.
SUBSTITUTE
=SUBSTITUTE(텍스트, 이전_텍스트, 새_텍스트, [인스턴스_번호])
텍스트에서 특정 문자열을 다른 문자열로 대체합니다.
예: =SUBSTITUTE(A1, "old", "new") - A1 셀의 "old"를 "new"로 대체합니다.

검색 및 참조 함수

VLOOKUP
=VLOOKUP(검색_값, 테이블_배열, 열_인덱스_번호, [범위_검색])
수직으로 데이터를 검색하여 해당하는 값을 반환합니다.
예: =VLOOKUP(A2, B2:D10, 3, FALSE) - A2의 값을 B2:D10에서 찾아 3번째 열의 값을 반환합니다(정확한 일치).
HLOOKUP
=HLOOKUP(검색_값, 테이블_배열, 행_인덱스_번호, [범위_검색])
수평으로 데이터를 검색하여 해당하는 값을 반환합니다.
예: =HLOOKUP(B1, A1:E5, 2, FALSE) - B1의 값을 A1:E5에서 찾아 2번째 행의 값을 반환합니다(정확한 일치).
INDEX
=INDEX(배열, 행_번호, [열_번호])
지정된 행과 열의 교차점에 있는 값을 반환합니다.
예: =INDEX(A1:C10, 5, 2) - A1:C10 범위에서 5번째 행, 2번째 열의 값을 반환합니다.
MATCH
=MATCH(검색_값, 검색_배열, [일치_유형])
검색 값의 상대적인 위치를 반환합니다.
예: =MATCH("사과", A1:A10, 0) - A1:A10에서 "사과"의 위치를 반환합니다(정확한 일치).
XLOOKUP
=XLOOKUP(검색_값, 검색_배열, 반환_배열, [일치_없을_경우], [일치_모드], [검색_모드])
강력한 검색 함수로, VLOOKUP/HLOOKUP의 향상된 버전입니다.
예: =XLOOKUP(A2, B2:B10, C2:C10) - A2의 값을 B2:B10에서 찾아 C2:C10에서 해당 값을 반환합니다.

날짜 및 시간 함수

TODAY
=TODAY()
현재 날짜를 반환합니다.
예: =TODAY() - 오늘의 날짜를 반환합니다(예: 2025-04-18).
NOW
=NOW()
현재 날짜와 시간을 반환합니다.
예: =NOW() - 현재 날짜와 시간을 반환합니다(예: 2025-04-18 14:30).
DATEDIF
=DATEDIF(시작_날짜, 종료_날짜, 단위)
두 날짜 사이의 차이를 계산합니다.
예: =DATEDIF(A1, B1, "d") - A1과 B1 사이의 일 수를 반환합니다.
DAY
=DAY(날짜)
날짜에서 일(day)을 추출합니다.
예: =DAY(A1) - A1 셀의 날짜에서 일 수를 반환합니다(예: 18).
MONTH
=MONTH(날짜)
날짜에서 월(month)을 추출합니다.
예: =MONTH(A1) - A1 셀의 날짜에서 월 수를 반환합니다(예: 4).
YEAR
=YEAR(날짜)
날짜에서 연도(year)를 추출합니다.
예: =YEAR(A1) - A1 셀의 날짜에서 연도를 반환합니다(예: 2025).

논리 함수

IF
=IF(논리_테스트, 참_값, 거짓_값)
조건을 테스트하여 참 또는 거짓에 따라 값을 반환합니다.
예: =IF(A1>100, "크다", "작다") - A1이 100보다 크면 "크다", 그렇지 않으면 "작다"를 반환합니다.
AND
=AND(논리1, [논리2], ...)
모든 조건이 참일 때 참을 반환합니다.
예: =AND(A1>10, B1<20) - A1이 10보다 크고 B1이 20보다 작으면 TRUE를 반환합니다.
OR
=OR(논리1, [논리2], ...)
조건 중 하나라도 참일 때 참을 반환합니다.
예: =OR(A1=10, B1=20) - A1이 10이거나 B1이 20이면 TRUE를 반환합니다.
NOT
=NOT(논리)
논리값을 반대로 변환합니다.
예: =NOT(A1=10) - A1이 10이 아니면 TRUE를 반환합니다.
IFERROR
=IFERROR(값, 오류_시_값)
수식이 오류를 반환하면 지정된 값을 반환합니다.
예: =IFERROR(A1/B1, 0) - A1/B1이 오류면 0을 반환합니다.
